# Employee-Salary-Analyzer-Pandas
A Python project built using Pandas to analyze employee data from a CSV file. The project generates salary insights, department-wise analysis, employee statistics, and a detailed employee analysis report.

## Features

- Display employee dataset
- Find the highest paid employee
- Find the lowest paid employee
- Calculate average salary by department
- Count employees in each department
- Find the highest salary
- Find the lowest salary
- Calculate average employee age
- Find the most experienced employee
- Find the youngest employee
- Find the oldest employee
- Display employees earning above average salary
- Display employees with more than 5 years of experience
- Show the top 3 highest paid employees
- Find the department with the highest average salary
- Generate a complete employee analysis report

## How to Run

1. Install Pandas

```bash
pip install pandas
```

2. Place `employees.csv` in the project folder.

3. Run the program

```bash
python main.py
```
